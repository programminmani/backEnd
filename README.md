# EmplManagerBackEnd

added some features to master
Simple NojeJs backend service for retrieving employees by position.

## Setup

The project requires npm and nojeJs installed

### Build Projects

1. Install typescript: `npm install -g typescript` 
2. Install npx: `npm install npx`
3. Install project dependencies: `npm install`

#### Launch server

1. `npx ts-node src/Server.ts`
2. Check localhost ap prot [8080](https://localhost:8080/)
